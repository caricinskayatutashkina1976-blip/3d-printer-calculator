# PrintCost Landing

Мини-проект с приветственным лендингом и калькулятором стоимости 3D-печати.

## Что внутри

- Одностраничный сайт с презентационным блоком
- Калькулятор себестоимости и цены продажи
- Пресеты материалов (PLA, PETG, ABS)
- Темная тема с сохранением в `localStorage`
- Копирование итогового результата в буфер обмена

## Быстрый старт

1. Откройте файл `3d_print_calculator.html` в браузере.
2. Заполните поля или нажмите кнопку `Демо-пример`.
3. Нажмите `Рассчитать`.

## Формула расчета

- Материал = `(цена_кг / 1000) * вес_модели`
- Работа принтера = `время_печати * цена_часа`
- Себестоимость = `материал + работа_принтера + дополнительные_расходы`
- Цена продажи = `себестоимость + (себестоимость * наценка / 100)`
